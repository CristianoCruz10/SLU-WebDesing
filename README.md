# SLU Web Design

Este repositório reúne as atividades desenvolvidas ao longo dos 8 módulos da disciplina de Web Design da Saint Leo University.

Os arquivos representam exercícios práticos realizados durante o curso, incluindo criação de páginas HTML, aplicação de estilos CSS, uso de imagens, edição de fotografia digital, imagem vetorial, áudio digital, formulários, tabelas e organização de conteúdo para web.

## Objetivo

O objetivo deste repositório é registrar o progresso das atividades feitas durante a disciplina e demonstrar a aplicação dos conceitos aprendidos em cada módulo.

## Organização

As atividades estão organizadas por módulo:

- `Módulo 1`: primeiros arquivos e estrutura básica de página web.
- `Módulo 2`: aplicação de estilos e uso de imagens.
- `Módulo 3`: projetos de fotografia digital, áudio digital e imagem vetorial.
- `Módulo 4`: desenvolvimento de página com conteúdo visual.
- `Módulo 5`: prática com HTML semântico e design.
- `Módulo 6`: formulários, tabelas e páginas de sistema.
- `Módulo 7`: conteúdo multimídia, galeria e créditos.
- `Módulo 8`: projeto final reunindo e personalizando arquivos das atividades anteriores.

## Observação

Este material foi produzido para fins acadêmicos, como parte das atividades da disciplina de Web Design da Saint Leo University.
