# Módulo 3 - Projetos digitais

Este diretório contém os três miniprojetos do Módulo 3:

- `fotografia.html`: projeto de fotografia digital com imagem editada.
- `audio.html`: projeto de áudio digital com arquivo `assets/ambience.wav`.
- `vetorial.html`: projeto de imagem vetorial com `assets/vector-art.svg` e `assets/vector-art.png`.

Abra `index.html` para navegar entre os projetos.
