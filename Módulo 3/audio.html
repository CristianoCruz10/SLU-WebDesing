Projeto de Áudio Digital

Arquivo de áudio original criado para uso em página web.

Detalhes do projeto

Este áudio é uma das melhores sinfonias de Beethoven, uma sinfonia ambiente leve e contínua. Ele pode ser usado como trilha sonora de fundo em uma página web ou apresentação multimídia.

O arquivo de áudio está disponível em https://www.youtube.com/watch?v=vCHREyE5GzQ&list=RDvCHREyE5GzQ&start_radio=1.

Abra em nova aba. Use com atribuição quando necessário.

Use-o em páginas web com o elemento <audio controls> ou exporte para outros formatos conforme necessário.
